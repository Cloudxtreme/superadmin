The User Roles

Resellers (outside of this scope)

  • Create new Cloudwalkers account (SoMe limit auto minus 3 accounts)
  • Delete Cloudwalkers account
  • Rename Cloudwalkers account

Settings

CAMPAIGN_CREATE Create campaigns
CAMPAIGN_DELETE Delete campaigns
SERVICE_CONNECT Connect new Social Media account (reduce limit by 1 if amount > 3)
SERVICE_DELETE Delete connected Social Media Account (add 1 to limit if new amount >= 3)
USER_INVITE Add user / invite user (to accout!)
USER_CREATE Add user (to account!)
USER_DELETE Delete user (from account!)
ACCOUNT_SETTINGS Change profile settings (incl change image)
USER_GRANT Edit the group of a user
GROUP_MANAGE Create or edit a user group.

Dashboard

DASHBOARD_VIEW View dashboard (foresee dashboard per user-role) For next dashboard - modules will have to be listed here

Messages (outgoing)

MESSAGE_SEND_COWORKERS User can create & send coworker messages
MESSAGE_SEND_INTERNAL User can send messages to the internal streams (push notifications)
MESSAGE_SEND_EXTERNAL User can send messages to social networks
MESSAGE_OUT_ATTACHMENTS User can add images to any messages
MESSAGE_OUT_ATTACHMENTS_INTERNAL User can add images to internal messages
MESSAGE_OUT_ATTACHMENTS_EXTERNAL User can add images to external messages
MESSAGE_OUT_ATTACHMENTS_COWORKERS User can add images to coworker messages messages
MESSAGE_OUT_SCHEDULE User can schedule messages
MESSAGE_OUT_REPEAT User can repeat a message
MESSAGE_OUT_EDIT User can edit all outgoing messages (as long as they are not sent yet, of course)
MESSAGE_OUT_EDIT_OWN User can only edit his own messages
MESSAGE_OUT_DRAFT User can create draft messages.

Messages (incoming)

MESSAGE_READ User can read all messages in the account
MESSAGE_ACTIONS User can execute any message actions

Inbox

MESSAGE_READ_INBOX User can read the inbox channel (everything = messages & notifications)
MESSAGE_READ_INBOX_MESSAGES User can read inbox messages.
MESSAGE_READ_INBOX_NOTIFICATIONS User can read inbox notifications
MESSAGE_READ_DRAFTS User can read drafts
MESSAGE_READ_SCHEDULE User can read scheduled messages.
MESSAGE_READ_SENT User can read sent messages.
MESSAGE_READ_FAILED User can read failed messages.
MESSAGE_READ_COWORKER User can read coworker messages

Company accounts

MESSAGE_READ_COMPANY Read messages from company account channel
MESSAGE_ACTIONS_COMPANY Act on messages in company account channel (note, if MESSAGE_ACTIONS is provided, this is always possible)

Accounts we follow

MESSAGE_READ_THIRDPARTY User can read messages from "accounts we follow" stream
MESSAGE_ACTIONS_THIRDPARTY Act on messages from "accounts we follow" (note, if MESSAGE_ACTIONS is provided, this is always possible)

Keyword monitoring

MESSAGE_READ_MONITORING Read messages from keyword monitoring channel
MESSAGE_ACTIONS_MONITORING Act on messages from "keyword monitoring" (note, if MESSAGE_ACTIONS is provided, this is always possible)
CHANNEL_MANAGE_ADD_MONITORING User can create keyword monitoring channels (categories & keywords)
CHANNEL_MANAGE_EDIT_MONITORING User can edit keyword monitoring channels
CHANNEL_MANAGE_DELETE_MONITORING User can delete keyword mointoring channels.

Statistics

STATISTICS_VIEW User can view statistics
