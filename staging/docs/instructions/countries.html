Countries

Editing a country midpoint and radius

If we want to edit an country, we can do it locally on the following URL:

	http://cloudwalkers-website.local/tests/countries.html?c=XX

Where "XX" is the country code (2 chars).

Example, to edit Portugal:

	http://cloudwalkers-website.local/tests/countries.html?c=pt

To see the final map:

	http://cloudwalkers-website.local/tests/countries.html
