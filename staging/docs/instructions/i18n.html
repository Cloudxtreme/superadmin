I18N using Polyglot

Nem implementation:

The new translation logic relies on a global reference to the Polyglot plugin, and should be applied inside views or templates.
To apply a translation inside a view: trans("string");

Or inside a template through Mustache:
{{#trans}}"string"{{/trans}}

String/translation format

The translation is not to be done through the use of a token, but through the actual English string:
Wrong:
trans("something_went_wrong");

Right:
trans("Something went wrong!");
If the current language is English, there will be no translation needed, hence, trans() will return the actual string.

The Locale translation files

Every language besides English will have a translation file, located in /locales/.
To add a translation, add the entry to the respective file: "Cheese omelette!" : "Omelette du fromage!"
