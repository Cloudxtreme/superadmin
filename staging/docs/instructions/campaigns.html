The custom campaign parameters

For campaign management, we currently add following fields to urls that are shortened:

  • utm_source: static string "Cloudwalkers"
  • utm_medium: The social network: Facebook, Twitter, LinkedIn, ...
  • utm_campaign: The provided campaign name
  • utm_content: "url" if sent in attachment, "message-body" if url is sent in message body

See Google Analytics for more information.
