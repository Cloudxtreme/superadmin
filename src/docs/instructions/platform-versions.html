Notes

A. 3 types of Notes
  • Contact notes
    Information available in current contact box. (latest messages, conversations, notes, tags)
  • Content notes
    Related to content from messages / notifications, keyword monitoring, company accounts, accounts we follow
    When we add a note to content, this content should be displayed with the note.
    When we add a note to content, the contact box of the owner of this content needs to be updated.
    When content is webscraped, the content should be copied and added to the note.
  • General notes
    Notes attached to the account.
B. Deliverables
  • Notes Page
    • A list overview of all notes
    • Filter on; contact, creation date, word search
    • Possibility to see all contacts with notes
    • Notes should be related-grouped (like inbox messages)
    • Create a new note
  • Positioned
    • Add a new note from dropdown menu
    • Message in inbox; read, edit and add note to the content of the message.
    • Notification in inbox; read, edit and add note to the content of the message.
    • Coworkers message on wall; read, edit and add note to the content of the message.
    • Company accounts; read, edit and add note to the content of the message.
    • Accounts we follow; read, edit and add note to the content of the message.
    • Keyword monitoring; read, edit and add note to the content of the message.
  • Extra
    Add function private or share note to a new note, default is private.
    Share is between 1) Webcare group members, 2) Conversation Managers.
C. Testing
  • Check if notes in contact box is updated
  • Add note to contact in contact box
  • Add note to content from a message/notification; check if content is updated on note page and on contact detail
  • Add note to content from company accounts / accounts we follow; check if content is updated on note page and on contact detail
  • Create a new Note from navigation and Note Page
  • Edit notes, check if information is correctly saved and added to contact box.
  • Delete note
  • Filter creation date results on Note Page
  • Search on contact name on Note Page
  • Filter a list of all contacts with notes on Note Page
  • Check if privacy mode is correctly exeuted between profiles
