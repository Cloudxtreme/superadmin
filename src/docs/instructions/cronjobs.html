cron/five-minutes.php

This cronjob updates the streams that require updates.

A stream requires updates if it is a high priority stream (updates every time the cronjob is called) OR when next_refresh date is in the past.

 

Every time a stream is updates, the system calculates the average amount of messages / hour and then estimates when a new message is to be expected. next_refresh is then set to that date.
