Emails

Reset Password

The email body structure is in engine under /app/views/emails/recover_password.blade.php

The email body is translated under folder /app/lang/en/emails/recover_password.php in variable "content" and the variables are passed as shown below.
This way is easier to change the text structure by language.

trans('emails/recover_password.content', array('firstname' => $firstname, 'codeurl' => $codeurl))

Events

API events controller

When some actions are made we may wanna create other actions associated to event like:

  • cleaning multiple things on delete and alert someone
  • on user register alert account managers
  • create background worker jobs like sending emails, generate user statistics, etc...
  • many others and must be applied to engine too 

		1. we register the listeners in app/events.php
		2. events and their actions are processed in app/controllers/EventsController.php
		3. trigger events anywhere with ex:
		   Event::fire('user.registration', array('msg', 'one more user registered in cloudwalkers'));
The translations for error validations are available @ /lang/en/validations.php

The variables is templates can be sanitized using {{{ $var }}}.
